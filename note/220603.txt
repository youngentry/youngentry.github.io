  </head>

sicky 구현

sticky 기본 구조입니다.
.sticky {
  position: sticky;
  top: 0;
  height: 100vh;
  overflow: hidden;
}
화면을 반반 나누는 구조를 만들고자 하면 flex: 1; 을 이용합니다.
.container {
  display: flex;
}

.container > * {
  flex: 1;
  display: flex;
  align-items: center;
}

우선 로고를 조절합니다.
header h1 {
  position: fixed;
  inset: 60px auto auto 60px;
  z-index: 999;
}

header img {
  width: 150px;
}

header i {
  position: fixed;
  inset: 60px 60px auto auto;
  z-index: 999;
}

쓰려는 속성이 인터넷에서 이용 가능한지 확인하는 사이트
https://caniuse.com/?search=inset




이미지를 한 가운데로 이동하는 방법입니다.
.section01 {
  display: flex;
  justify-content: center;
  align-items: center;
}
