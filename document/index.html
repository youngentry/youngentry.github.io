position

  1. position의 기초

    display 의 속성 none, block, inline-bloxk, inline, flex 을 이해합니다.

    position 의 속성 static, relative, absolute, fixed 을 이해합니다.

  2. position: pop-up 창 만들기

    팝업창을 통해 포지션을 배웁니다.

    transform: translate(-50%, -50%); transform은 layout을 깨지 않습니다. 자기 자신을 기준으로 합니다.

    팝업창 기본 형태는 외웁니다. .popup { position: absolute; top:50%; left: 50%; transform: translate(-50%, -50%); width: 450px; height: 350px; background: red; }

    addEventListener를 이해합니다. .addEventListener('click', function())

    querySelector를 이해합니다. document .querySelector(".popup button") .addEventListener("click", function () { console.log("done"); document.querySelector(".popup").style.display = "none"; });

    display:none(과 같이 수치가 없는, 중간단계가 없는 속성)은 trasition이 적용 안됩니다.

    양방향을 지정하면 width를 지정하지 않아도 사이의 간격 만큼 span의 width가 나타나게 됩니다. left: 5px; right: 5px;
